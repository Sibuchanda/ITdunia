Atlassian

About Company: Atlassian is an Australian software company known for developing products for software development, project management, and collaboration, such as Jira, Confluence, and Bitbucket.

CEO: Scott Farquhar & Mike Cannon-Brookes

Headquarters: Sydney, Australia

Founded: 2002

Industry: Software Development, Collaboration Tools

Eligibility Criteria

  • Bachelor's or Master's degree in Computer Science, IT, or related fields
  • Strong programming skills in Java, Python, or JavaScript
  • Experience with DevOps, cloud computing, and CI/CD tools
  • Problem-solving and analytical thinking

Placement Process

  1. Online Coding Assessment
  2. Technical Interviews (DSA, Software Design, and Problem-Solving)
  3. Behavioral Interview (Culture fit and communication skills)
  4. Final Offer

Hiring Exam

Atlassian evaluates candidates through technical coding rounds, system design discussions, and DevOps-related assessments.
