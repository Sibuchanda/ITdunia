Microsoft

About Company: Microsoft is a leading multinational technology company specializing in software, cloud computing, artificial intelligence, and hardware. It is known for products like Windows, Office, Azure, and Xbox.

CEO: Satya Nadella

Headquarters: Redmond, Washington, USA

Founded: 1975

Industry: Software, Cloud Computing, AI, Consumer Electronics

Eligibility Criteria

  • B.E / B.Tech / M.E / M.Tech in Computer Science, IT, or related fields
  • MCA / M.Sc in Computer Science or IT
  • Strong DSA, problem-solving, and software development skills
  • Good knowledge of algorithms, operating systems, and cloud computing
  • Strong communication and analytical skills

Placement Process

  1. Online Coding Assessment(DSA, Algorithms, Problem-Solving)
  2. Technical Interviews (2-3 Rounds) Covering Data Structures, System Design
  3. HR Interview Behavioral and situational questions
  4. Final Offer**

Hiring Exam

Microsoft conducts the **Microsoft Online Assessment (OA), which includes coding, aptitude, and system design questions.
